## 기능 1️⃣: KDC 마인드맵 탐색

### 📌 목표
학생이 "도서 탐색" → "자연과학" → "물리학" → "고전역학" 순서로 내려가며 분야를 탐색

### 🔍 공공데이터 출처
**교육청 교육공공데이터 - 표준분류체계.xlsx**
- 한국십진분류법(KDC) 10대 분류 (000~900)
- 각 분류의 중분류, 소분류 계층 구조

### 📊 데이터 형태

**입력 (원본)**:
```
표준분류체계.xlsx 의 "분류" 시트
─────────────────────────────────────
분류코드 | 분류명      | 부분류
─────────────────────────────────────
000     | 총류        | (없음)
010     | 도서관학    | 000의 자식
020     | 컴퓨터      | 000의 자식
...
400     | 자연과학    | (없음)
410     | 수학        | 400의 자식
420     | 물리학      | 400의 자식
430     | 화학        | 400의 자식
...
```

**출력 (가공됨) - `kdc_tree.json`**:
```json

  "code": "",

          "name": "컴퓨터",

          "children": []
        }
      ]
    },
    {
      "id": "400",
      "name": "자연과학",
      "code": "400",
      "children": [
        {
          "id": "410",

          "code": "410",
          "children": []
```

### 🛠️ 가공 방법
1. **표준분류체계.xlsx 읽기** → 분류코드, 분류명 추출
2. **계층 관계 파악** → "3자리 코드"는 10대 분류, "자리수 더 있으면" 세부 분류
3. **JSON 트리 구조로 변환** → parent-children 관계 설정
4. **결과 저장** → `kdc_tree.json`

## 기능 2️⃣: 책 카드 표시 + 학과 연결

### 📌 목표
선택된 분야의 도서 정보 + 학과 + 진로를 한 카드에 표시

### 🔍 공공데이터 출처 (2가지)

#### 데이터 A: 사서추천도서 API
**국립어린이청소년도서관 사서추천도서 공공데이터**
- 출처: 공공데이터포털 또는 API 직접 호출
- 포함 정보: 도서명, 저자, 출판사, 도서소개, KDC, 발행연도

**예시 (API 응답)**:
```json
{
  "title": "Love That Dog : 아주 특별한 시 수업",
  "author": "샤론 크리치",
  "publisher": "시공사",
  "description": "잭은 시는 여자애들이나 쓰는 것이라고 생각하던 소년...",
  "kdc": "800",
  "reg_date": "2024-01-15"
}
```

#### 데이터 B: 전국대학별학과정보표준데이터
**교육청 공공데이터 - 전국대학별학과정보표준데이터.csv**
- 출처: 공공데이터포털
- 포함 정보: 학과명, 주요교과목명, 관련직업명, 학과상태, 학과별코드
- 크기: 약 5만 행 (전국 대학 모든 학과)

**예시 (CSV 원본)**:

```javascript
function renderBookCard(book) {
  // 1. 기본 정보 채우기
  document.getElementById('book-title').textContent = book.title;
  document.getElementById('book-author').textContent = book.author;
  
  // 2. 태그 표시

  book.student_tags.forEach(tag => {

    span.className = 'tag';
    span.textContent = '#' + tag;
    tagList.appendChild(span);
  });
  
  // 3. KDC 분야명 표시
  let careerInfo = careerMap[book.kdc];  // career_map.json에서 룩업

    '분야: ' + careerInfo.kdc_name + ' (' + book.kdc + ')';
  
  // 4. 🔑 학과 정보 표시 (상위 3개)
  let deptList = document.getElementById('dept-list');
  careerInfo.departments.slice(0, 3).forEach(dept => {

    div.className = 'department';
    div.innerHTML = `

      <p class="subjects">배우는 것: ${dept.subjects.slice(0, 3).join(', ')}</p>
      <p class="careers">진로: ${dept.careers.slice(0, 2).join(', ')}</p>

    deptList.appendChild(div);
  });
}
```

## 기능 3️⃣: "다른 시각으로 보기" (옆 서가 이동)

- 각 KDC별로 의미 있는 연결 3개씩 설계
- 학문 체계와 진로 탐색 경험을 고려해서 작성

### 📊 데이터 형태

**JavaScript**:
```javascript
function renderOtherView(currentKdc) {
  // 1. side_shelf_map에서 현재 KDC의 연결 가져오기
  let connections = sideShelfMap.connections[currentKdc];
  if (!connections) return;
  
  // 2. 3개 중 1개 랜덤 선택
  let target = connections[Math.floor(Math.random() * 3)];
  
  // 3. target_kdc와 example_tags로 책 찾기
  let recommendedBooks = books_tagged
    .filter(b => b.kdc === target.target_kdc)
    .filter(b => {
      // 태그 매칭: example_tags와 student_tags의 교집합
      return b.student_tags.some(tag =>
        target.example_tags.some(ex => 
          tag.includes(ex) || ex.includes(tag)
        )
      );
    })
    .slice(0, 3);  // 상위 3개
  
  // 4. HTML 렌더링
  let html = `
    <div class="view-option">
      <h4>${target.label}</h4>
      <p class="description">${target.reason}</p>
      <div class="recommended-books">
        ${recommendedBooks.map(b => `
          <div class="mini-card" onclick="showBook('${b.title}')">
            <h5>${b.title}</h5>
            <p>${b.author}</p>
          </div>
        `).join('')}
      </div>
    </div>
  `;
  
  document.getElementById('other-view-content').innerHTML = html;
}
```

## 기능 4️⃣: 개인 독서 분석

### 📌 목표
학생이 읽은 책들 → 분야 비율 시각화 → "과학 분야 많음, 철학 적음" 인사이트 제공

### 🔍 공공데이터 출처
**books_tagged.json만 사용** (외부 공공데이터 없음)

### 📊 데이터 처리

**입력**: 
```javascript
// localStorage에 저장된 사용자의 읽은 책 인덱스들
localStorage.myBooks = JSON.stringify([0, 5, 23, 127, ...]);
```

**처리**:
```javascript
// 1. 읽은 책 가져오기
let myBookIndices = JSON.parse(localStorage.myBooks);
let myBooks = myBookIndices.map(idx => books_tagged[idx]);

// 2. KDC별로 그룹핑
let kdcCount = {};
myBooks.forEach(book => {
  let kdc = book.kdc;
  kdcCount[kdc] = (kdcCount[kdc] || 0) + 1;
});
// 결과: {"400": 8, "800": 5, "500": 3, ...}

// 3. 태그 수집 + 빈도 계산
let tagFreq = {};
myBooks.forEach(book => {
  book.student_tags.forEach(tag => {
    tagFreq[tag] = (tagFreq[tag] || 0) + 1;
  });
});
// 결과: {"시쓰기": 5, "감정표현": 3, "성장": 4, ...}
```

```javascript
// 미탐색 분야 추천
function recommendUnexploredKdcs(kdcCount) {
  let allKdcs = ['000', '100', '200', '300', '400', '500', '600', '700', '800', '900'];
  let explored = Object.keys(kdcCount);
  let unexplored = allKdcs.filter(k => !explored.includes(k));
  
  let kdcNames = {
    '100': '철학', '200': '종교', '300': '사회과학',
    '400': '자연과학', '500': '기술', '600': '예술',
    '700': '언어', '800': '문학', '900': '역사'
  };
  
  let recommendation = unexplored
    .map(k => kdcNames[k])
    .slice(0, 3)
    .join(', ');
  
  document.getElementById('rec-text').textContent = 
    `${recommendation} 분야를 탐색해보세요!`;
}
```
